The recency-reducibility gate

Two detectors read the same generated data. The naive separability test asks “are train and val different?” — which a forecast holdout answers yes by construction. The gate asks the only question sliding can act on: does the recent block actually move closer to val? Every verdict below is computed live by gate.js.
